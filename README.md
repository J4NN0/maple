# Maple

Website for Maple, a toy poodle with too much energy and her own Instagram ([@mapleandchill](https://www.instagram.com/mapleandchill/)). The site lets people play mini-games where Maple is the star, and links back to her IG page.

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
